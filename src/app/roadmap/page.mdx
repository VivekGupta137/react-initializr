# Roadmap

## Features

- users can search for starter template
- could add their own template to the list - the user will need to provide the url to the package.json file
- users can search for a specific package
- users can select specific packages
- can select either react / vue / quik / svelte
- can select specific version of the package

- filter
    - Framework : React / Vue / Quik / Svelte / All (any framework)
    - CSS : Tailwind / Bootstrap / MaterialUI / chakraUI / CSS in JS / All (any css)
    - ORM : Prisma / TypeORM / Sequelize / All (any orm)
    - Database : Postgres / MySQL / SQLite / MongoDB / All (any database)
    - Auth : Passport / JWT / Firebase / Auth0 / All (any auth)
    - Testing : Jest / Mocha / Cypress / All (any testing)
    - CI/CD : Github Actions / Gitlab CI / Travis CI / All (any ci/cd)
    - module bundler : Vite / Webpack / Snowpack / All (any bundler)
    - state management : Redux / Mobx / Recoil / Zustand / All (any state management)
    - form library : Formik / React Hook Form / Final Form / All (any form library)
    - use case : Blog / E-commerce / Social Media / All (any use case)
    - language : TypeScript / JavaScript / All (any language)

    after selection of specific filter, let the user select specific version as well


- sort
    - stars
    - downloads
    - last updated
    - created date
    - name
    - version

- pagination
- dark mode
- light mode
- responsive
- mobile first
